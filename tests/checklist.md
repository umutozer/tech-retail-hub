# Kalite Kontrol Listesi
## HTML Validasyonu
- [ ] Tüm HTML etiketleri doğru kapandığından emin olun.
- [ ] Tüm HTML etiketleri semantic olarak doğru kullanıldığından emin olun.
## CSS Uyumluluğu
- [ ] Tüm CSS stilleri doğru uygulandığından emin olun.
- [ ] Tüm CSS stilleri responsive tasarım için doğru yapılandırıldığından emin olun.
## JS Hataları
- [ ] Tüm JavaScript kodları hata olmadan çalıştığından emin olun.
- [ ] Tüm JavaScript kodları güvenlik açıklarına karşı korumalı olduğunda emin olun.
## Responsive Tasarım
- [ ] Tüm sayfalar mobil, tablet ve masaüstü cihazlarda doğru görüntülendiğinden emin olun.
## SEO
- [ ] Tüm sayfaların meta etiketleri doğru yapılandırıldığından emin olun.
- [ ] Tüm sayfaların heading hierarchy doğru yapılandırıldığından emin olun.
- [ ] Tüm görsellerin alt textleri doğru yapılandırıldığından emin olun.
## Güvenlik
- [ ] Tüm forma bağlı işlemlerde CSRF koruması uygulanmalıdır.
- [ ] Tüm kullanıcı girişlerinde XSS koruması uygulanmalıdır.
## Performans
- [ ] Tüm sayfaların yüklenme hızının 3 saniyeden az olduğu doğrulanmalıdır.
- [ ] Tüm görsellerin boyutları optimize edildiğinden emin olun.
## Erişilebilirlik
- [ ] Tüm sayfaların WCAG 2.1 AA standartlarına uygun olduğu doğrulanmalıdır.
- [ ] Tüm sayfaların renk kontrastının yeterli olduğu doğrulanmalıdır.
